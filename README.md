## how to use

## summary
+ pace.py : Calculate the pace in min/km.

## complete list
### pace.py

```
usage: pace [-h] -t TIME -d DISTANCE

Calculate the pace in min/km.

optional arguments:
  -h, --help            show this help message and exit
  -t TIME, --time TIME  Time in mm:ss
  -d DISTANCE, --distance DISTANCE
                        Distance in metres
```
